## Important Notes
- Only `@vitbhopal.ac.in` emails can register
- OTP emails delivered instantly via Resend (zero Gmail/SMTP issues)
- Backend on Render free tier → first request may take 10–30 sec after inactivity (cold start). Normal & will be upgraded on official adoption.

## Core Features Delivered
### Member Portal
- Only `@vitbhopal.ac.in` emails allowed (strict domain enforcement)
- Real OTP verification via Resend (instant & beautiful emails)
- Secure JWT + httpOnly cookies
- Responsive glassmorphism UI (Tailwind + shadcn/ui)
- Personal member dashboard with profile & upcoming events

### QR Ticketing & Verification System
- **Organizer Dashboard** – Add participants & generate unique QR tickets
- **Secure QR tickets** with signed payload (ID + signature)
- **Live verification page** – Scan QR to validate ticket
- **Automatic "Used" marking** after successful scan (prevents reuse)
- Fully integrated with MongoDB Atlas


## Tech Stack & Deployment
| Layer             | Technology                                 | Hosted On     |
|-------------------|--------------------------------------------|---------------|
| Frontend          | React 19 + Vite + Tailwind + shadcn/ui    | Vercel        |
| Backend           | Node.js + Express + Mongoose               | Render        |
| Database          | MongoDB Atlas                              | MongoDB Cloud |
| Auth              | JWT + httpOnly cookies                     | Secure        |
| Email (OTP)       | Resend                                     | Resend        |
| QR Generation     | `qrcode` + signed payload                  | Backend       |
